# Rock Paper Scissors Game
A simple console-based Rock Paper Scissors game developed in C.

## Features
* Play against the computer
* Random system-generated moves
* Score tracking
* First player to reach 3 wins is declared the winner
* Console-based user interface

## How It Works

1. Choose:
   * Rock
   * Paper
   * Scissor

2. The computer randomly selects its move.

3. The winner of each round is determined based on the standard rules:
   * Rock beats Scissor
   * Scissor beats Paper
   * Paper beats Rock

4. The game continues until either the player or the computer reaches 3 wins.
